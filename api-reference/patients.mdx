## Endpoints

- <span class="badge post">POST</span> [`CreatePatientProfile`](endpoint/post-createpatientprofile)  
  Creates a new patient profile.

- <span class="badge post">POST</span> [`FetchMyPatients`](endpoint/post-fetchmypatients)  
  Fetches a paginated list of patients associated with the authenticated organization's API key.
