## Overview

Tesa Cloud uses FHIR R4 [extensions](http://hl7.org/fhir/R4/extensibility.html) to represent surgical data that has no standard FHIR equivalent — such as graft specifications, fixation devices, and intraoperative tension readings.

All extensions use the base URL:

```
https://tesacloud.com/fhir/StructureDefinition/{extension-name}
```

Extensions appear on **Procedure** resources and can be read on search/read and written via `PUT /fhir/r4/Procedure/{id}`.

## Procedure Extensions

### Graft

| Extension | Value Type | Description |
|---|---|---|
| `graft-type` | CodeableConcept | Graft type (SNOMED coded — see below) |
| `graft-origin` | string | Origin tissue |
| `graft-length` | decimal | Length in mm |
| `graft-diameter` | decimal | Diameter in mm |
| `hamstring-type` | string | Hamstring graft variant |
| `quad-type` | string | Quadriceps graft variant |
| `btb-type` | string | Bone-tendon-bone variant |

### Fixation

| Extension | Value Type | Description |
|---|---|---|
| `femur-fixation` | string | Femoral fixation device |
| `femur-drilling-technique` | string | Femoral drilling technique |
| `femur-socket-size` | decimal | Femoral socket size (mm) |
| `tibia-fixation` | string | Tibial fixation device |
| `tibia-drilling-technique` | string | Tibial drilling technique |
| `tibia-socket-size` | decimal | Tibial socket size (mm) |
| `tibia-backup-fixation` | string | Tibial backup fixation |

### Tension

| Extension | Value Type | Description |
|---|---|---|
| `pre-tension-force` | decimal | Pre-tension force (Newtons) |
| `pre-tension-duration` | integer | Pre-tension duration (seconds) |
| `fixated-tension-graft` | decimal | Final graft tension at fixation (Newtons) |
| `fixated-tension-brace` | decimal | Brace tension at fixation (Newtons) |

### Tunnel

| Extension | Value Type | Description |
|---|---|---|
| `femoral-tunnel-position` | string | Clock position of femoral tunnel |
| `femoral-tunnel-angle` | decimal | Tunnel angle (degrees) |

### Augmentation

| Extension | Value Type | Description |
|---|---|---|
| `augment-type` | string | Augmentation type |
| `augment-femoral-fixation` | string | Femoral augment fixation |
| `augment-tibial-fixation` | string | Tibial augment fixation |

### Other

| Extension | Value Type | Description |
|---|---|---|
| `tourniquet-time` | string | Tourniquet time |

## Extension Example

A Procedure resource with surgical extensions:

```json
{
  "resourceType": "Procedure",
  "id": "7",
  "status": "completed",
  "code": {
    "coding": [{
      "system": "http://snomed.info/sct",
      "code": "31154006",
      "display": "Anterior cruciate ligament structure"
    }]
  },
  "subject": { "reference": "Patient/42" },
  "encounter": { "reference": "Encounter/15" },
  "extension": [
    {
      "url": "https://tesacloud.com/fhir/StructureDefinition/graft-type",
      "valueCodeableConcept": {
        "coding": [{
          "system": "http://snomed.info/sct",
          "code": "21753001",
          "display": "Autogenous graft"
        }]
      }
    },
    {
      "url": "https://tesacloud.com/fhir/StructureDefinition/graft-length",
      "valueDecimal": 28.5
    },
    {
      "url": "https://tesacloud.com/fhir/StructureDefinition/femur-fixation",
      "valueString": "Endobutton CL"
    },
    {
      "url": "https://tesacloud.com/fhir/StructureDefinition/fixated-tension-graft",
      "valueDecimal": 80.0
    }
  ]
}
```
